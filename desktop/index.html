Markee Desktop
The Markdown editor to create document and slideshow using Markdown.

Features

create and edit files in local machine You can create new markdown file and edit existing markdown file in the application
render markdown
Markdown syntax is supported using Showdown
create presentation Presentations are created using a library forked from the Remarkjs library
fullscreen display Fullscreen slideshow or document display can be launched directly within the application
other features Other features include presenter mode for slideshow, cloning of display, display sync between cloned and main display, content sync from main window to cloned display, …

Markee Web

use Markee web to create and modify Markdown files on Google Drive
